# RASPBERRY WEB INFRASTRUCTURE
Self-hosted, low-cost, and energy-efficient web infrastructure built on Raspberry Pi using Apache and Cloudflare Tunnel. Ideal for startups and personal projects seeking privacy, stability, and full control without relying on third-party servers.

---

## ⚙️ Key Features
- 🧠 **Low-power infrastructure:** Built on ARM hardware for minimal energy use.  
- 🔒 **Secure exposure:** Cloudflare Tunnel provides encrypted traffic without opening ports.  
- 🧰 **Lightweight setup:** Based on Raspberry Pi OS Lite with Apache as the core web server.  
- 🌐 **Self-hosted independence:** Full control over your data and deployment.  
- 🧱 **Scalable base:** Extendable to local services or internal networks.  

---

## 🧠 Project Goals
- Demonstrate **efficient infrastructure design** using open-source tools.  
- Provide a **functional, low-cost alternative** for static or semi-dynamic websites.  
- Encourage learning in **Linux administration, networking, and self-hosting**.  
- Serve as a **base architecture** for small business or educational environments.
